## DXF Engraving Tool

Инструмент для автоматического заполнения (штриховки) замкнутых контуров в DXF файлах. 
Идеально подходит для подготовки макетов под лазерную гравировку (нержавеющая сталь, таблички).

### Возможности:
- Автоматическая очистка и "лечение" поврежденной геометрии (Topology Correction).
- Поддержка вложенных контуров (отверстия в буквах).
- Соосная генерация линий для быстрой работы станка.

### Как пользоваться:
1. Установите зависимости:
   `pip install ezdxf shapely`
2. Запустите скрипт через терминал:
   `python DXF_Engrave.py "путь/к/файлу.dxf" --step 0.5`

**Параметры:**
- `input`: Путь к вашему чертежу.
- `--step`: Шаг между линиями в мм (по умолчанию 1.0).
